## Terms

- `<machine-host>` is the SSH target you pass to `yeet init`, such as
  `root@server.example`.
- `<catch-host>` is the tsnet hostname yeet uses after install. Use it with
  `CATCH_HOST`, `--host`, or `<svc>@<catch-host>`.
- `<svc>` is a service name.
- `<file>` is a local file path.
