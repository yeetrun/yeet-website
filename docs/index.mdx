Yeet runs from your workstation and asks catch to do the boring host work:
install services, move files, start units, tail logs, and clean up after itself
when you tell it to. The point is not to hide Linux. The point is to keep the
state visible enough that you can debug it when Friday arrives with opinions.

Start with setup. Then pick the guide for the thing you actually want to run.
