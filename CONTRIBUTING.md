# Contributing to the Yeet Docs Site

This repository contains the documentation site for **yeet** and **catch**.

## Quick Guide

- **Fix a typo or clarity issue:** open a PR directly.
- **Bug in the website:** open an issue with clear repro steps.
- **New docs or feature request:** open a discussion in the main yeet repo first.

Docs content lives in `docs/`. Site pages live in `src/pages/`.
